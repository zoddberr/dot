# dot

I decided to design, maintain and use my system with the help of YAGNI principle. \\
This principle usually refers to programming, but I think it could be also used as system "design" helper, especially in unix. \\
Basically I won't add additional functionality in my system unless I *truly* need it. Real minimalism.

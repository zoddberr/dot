# dot

I decided to design, maintain, and use my system with the help of the YAGNI principle.  
This principle usually refers to programming, but I think it could also be used as a system "design" helper, especially in unix.  
Basically, I won't add additional functionality to my system unless I *truly* need it. Real minimalism.
